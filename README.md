# Frontend Mentor - Componente QR Code

![Design preview for the QR code component coding challenge](./preview.jpg)

## Bem-vindo(a)!

Obrigado por conferir este desafio de codificação front-end.

## O Desafio

O objetivo deste desafio foi construir um **componente de QR Code** e fazê-lo se parecer o máximo possível com o design fornecido.

Para este projeto, é necessário um entendimento básico de **HTML e CSS**.

## Estrutura do Projeto

* Os arquivos de **design** (versões mobile e desktop) estão localizados na pasta `/design`.
* Todos os **assets** necessários estão na pasta `/images`.
* Um guia de estilo (incluindo paleta de cores e fontes) está disponível no arquivo `style-guide.md`.

## Tecnologias Utilizadas

* HTML5
* CSS3

## Como Iniciar

1.  Clone este repositório.
2.  Abra o arquivo `index.html` no seu navegador.
3.  Utilize o `style-guide.md` e os designs para implementar o layout e os estilos.

**Divirta-se codificando!**
